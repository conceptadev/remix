`RemixSpinner` renders an animated eight-leaf loading indicator. It can stand
alone or replace a child while preserving that child's layout.

## When to use this

- **Pending actions**: Replace button content while work is in progress
- **Async content**: Indicate an operation with no measurable completion value
- **Stable layouts**: Keep the original child's dimensions while loading

## Basic implementation

<CodeGroup title="Loading replacement" defaultLanguage="dart">
```dart
import 'package:flutter/material.dart';
import 'package:remix/remix.dart';

class SpinnerExample extends StatelessWidget {
  const SpinnerExample({super.key, required this.loading});

  final bool loading;

  @override
  Widget build(BuildContext context) {
    return RemixSpinner(
      loading: loading,
      semanticLabel: 'Saving',
      style: RemixSpinnerStyler()
          .size(20)
          .color(Colors.blue)
          .opacity(0.7)
          .leafRadius(const Radius.circular(2))
          .duration(const Duration(milliseconds: 800)),
      child: const Text('Save changes'),
    );
  }
}
```
</CodeGroup>

When `loading` is false, `child` is returned normally. When it is true, the
child is hidden but still participates in layout and the spinner is centered
over its footprint. With no child, the spinner uses its configured square size.

## Constructor

<CodeGroup title="RemixSpinner constructor" defaultLanguage="dart">
```dart
const RemixSpinner({
  Key? key,
  bool loading = true,
  Widget? child,
  String? semanticLabel,
  bool excludeSemantics = false,
  RemixSpinnerStyler style = const RemixSpinnerStyler.create(),
  RemixSpinnerSpec? styleSpec,
})
```
</CodeGroup>

## Behavior and semantics

- `loading: false` returns `child`, or an empty box when no child exists.
- `loading: true` exposes the loading-spinner role unless `excludeSemantics` is
  true. Supply `semanticLabel` when context does not already name the operation.
- When reduced motion is requested, animation stops at a stable frame.
- Spinner color defaults to the inherited foreground color, so it composes with
  Button, IconButton, and other styled content.

## Styling

`RemixSpinnerStyler` exposes `size`, `color`, `opacity`, `leafRadius`, and
`duration`. The spinner always uses eight fading leaves; ring-track and stroke
properties from the former circular painter are not part of this API.
